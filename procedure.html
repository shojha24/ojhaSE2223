1

Gather materials.

You need: A computer (Dedicated GPU is preferable), audio and image datasets of cats and dogs from Kaggle, and Python (libraries to import include tensorflow, numpy, matplotlib)

2

Write code.

We have a dataset of cat/dog images, and cat/dog audio. Using a special algorithm known as a CNN, which makes maps of images and audio, the code can find features in the data, and can then use it to differentiate cats and dogs. Over multiple test runs, the code gets better at this task.

3

Execute code.

After testing the algorithm, the final accuracy of the code is averaged out and given to the researcher. This final amount would show how accurate the model has become after consistent training, and would serve as a data point.

4

Perform data analysis.

Both the image and the audio algorithm are tested 12 times. This added up to 24 total trials, or 24 data points in the raw data table, which would later be analyzed through a summative data table, a calculated data table, and a statistical data table, each depicting how varied the final accuracies were.
